Flowchart Studio ProfessionalJavaFXで構築された、直感的かつ軽量なフローチャート作成エディタです。シンプルな操作で、システム設計、アルゴリズムの可視化、業務フロー図を素早く作成し、画像として出力したりデータとして保存したりすることができます。🚀 主な機能1. 直感的なダイアグラム作成多彩なノード形状: 開始/終了（楕円）、処理（長方形）、判断（ひし形）の3種類をサポート。簡単接続: 右クリック操作だけでノード間を矢印で連結。グリッド・スナップ: ノードを動かすと自動的に20px間隔のグリッドに吸着し、整列が容易です。2. 高度なエディタ機能動的ラベル: 接続線の上に「Yes/No」などのラベルを配置。線が動いても常に中央に自動追従します。ズーム機能: Ctrl + マウスホイール で、広大なキャンバスを自在に拡大縮小（20% ～ 500%）。右クリックメニュー: テキスト編集、色の変更、線の反転、削除などの主要操作に即座にアクセス。3. データ管理と出力JSON保存/読込: 作成したチャートをJSON形式で保存し、いつでも再編集が可能。PNGエクスポート: 完成した図を画像ファイルとして書き出し、報告書やプレゼン資料に活用できます。ショートカット: Deleteキーでの削除や、Ctrl + D によるノードの複製に対応。🛠 インストールと実行方法動作環境Java: JDK 17 以上JavaFX: 17 以上実行手順このリポジトリをクローンするか、ソースコードをダウンロードします。JavaFXのライブラリ（controls, swing, graphics等）をクラスパスに含めてコンパイルします。FlowchartStudio クラスを実行してください。Bash# コンパイル例（環境に合わせてパスを調整してください）
javac --module-path $PATH_TO_FX --add-modules javafx.controls,javafx.swing,javafx.fxml FlowchartStudio.java

# 実行
java --module-path $PATH_TO_FX --add-modules javafx.controls,javafx.swing FlowchartStudio
